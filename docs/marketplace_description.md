Product name
Glassbox Preflight Certifier™

Subtitle
Input Compatibility Certification for the Glassbox Pipeline

One-liner
Certifies your inputs against the Glassbox Pipeline’s required schema and formats, then generates a versioned Ready-to-Run Pack to prevent failed runs.

Above-the-fold bullets (4–6)

Schema-locked validation: checks only the exact inputs/config the Glassbox Pipeline supports

PASS / WARN / FAIL: clear gate result plus remediation steps

Ready-to-Run Pack: produces a deterministic bundle the pipeline can consume reliably

Audit-grade provenance: input hashes, schema version, tool versions, compatibility tag

Runs in your project: reads/writes only to your specified Cloud Storage locations (if true)

Defensible “certification” phrasing (use this exact line)
“Certification covers schema and format compatibility with the Glassbox Pipeline — not scientific validity or outcome guarantees.”Product name
